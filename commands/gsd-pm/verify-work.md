# Command Removed: /gsd-pm:verify-work

This command was part of the execution workflow, which has been removed.

**GSD is now planning-only.** Plans created with `/gsd-pm:plan-phase` are implementation specifications for your engineering team, not executable instructions for Claude.

## What to do instead

- **Plan a phase:** `/gsd-pm:plan-phase {N}`
- **Mark phase complete:** `/gsd-pm:complete-phase {N}` (auto-advances to next phase)
- **Revise planning artifacts:** `/gsd-pm:edit-phase {N}`
- **View available commands:** `/gsd-pm:help`
